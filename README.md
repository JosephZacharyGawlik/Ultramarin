# Ultramarin
Project for the Ultramarin Data Science Practical WS 25/26.

[Google Drive](https://drive.google.com/drive/u/0/folders/15bYTvG7qebiivza142Q3cBmxjd0jxOyJ) for Literature & Data.

Setup for uv:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh # download uv

source $HOME/.local/bin/env # add uv to PATH

uv sync # create environment

source .venv/bin/activate # activate environment
```
